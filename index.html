HyperCard UI Kit

HyperCard UI Kit (Static Scaffolds)

This kit provides static, accessible HTML scaffolds for core HyperCard UX surfaces (card window, script editor, message box, debugger, inspectors, dialogs, and more). Wire in your runtime to make them live.

Card View Background Editor Script Editor Message Box Debugger Inspectors Find & Mark Ask / Answer Printing & Reports Preferences Apple Events Window Management Sound & Paint

Notes

  • All pages share the same menu bar, tool palette, and message box.
  • Use data- attributes and IDs to bind your JavaScript runtime.
  • Typography is system fonts only; add your own theme as needed.
